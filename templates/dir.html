Instructions

  • From drakahari.com, select the apply link on any of the open positions.
    • Some fields are required: Full Name, Email, Linkedin (must use a URL)
  • If submitted properly, a confirmation screen will be displayed.
  • Now select the Review App Status link from the footer section @ drakahari.com.
    • You should now see that your application information has been added to the queue of applications!
  • Now to test admin functionality by adding additional positions the home screen.
    • Select Admin in the footer section of drakahari.com.
    • This is password protected and the username is admin and the password is admin (kept simple for testing purposes). Please feel free to use incognito mode if you want to make multiple entries to the database.
    • After entering all of the relevant job data, hit submit.
    • You will now see a screen that displays Job added successfully! Check on drakahari.com to see if the position was in fact added.
  • From drakahari.com, select the To-Do Demo link that is in the footer. This will take you to a page that has a simple To-Do list that utilized MongoDD NOSQL database
    • Add items to the To-Do list and notice that they are added below.
    • You can then delete items from the To-Do list to show that there is dynamic two-way communication between the application and the NOSQL database.
